# PoePriceRunesOfAldurHelper-ru

Оверлей для Path of Exile 2, который в реальном времени распознаёт русский текст с экрана 
(встроенный Windows OCR) и показывает цену каждого предмета в Divine / Chaos / Exalted 
с иконками валют.

## Возможности

- **F7** — выделить область экрана для распознавания (сохраняется между запусками)
- **F8** — запуск / остановка непрерывного распознавания (каждые 1.5 сек)
- **F6** — обновить цены с poe.ninja (автоматически при запуске тоже)
- Цены выводятся напротив каждого распознанного предмета
- Учитывается количество: «Божественная сфера (10)» → цена ×10
- Самая дорогая позиция подсвечивается золотым на красном фоне
- Клики проходят сквозь оверлей — не мешает управлению в игре
- Результаты пишутся в `recognition_log.txt`

## Системные требования

- **Windows 10/11** (сборка 19041+)
- **.NET 10 Desktop Runtime** — [скачать](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)
- **Path of Exile 2** в режиме **borderless window** (не exclusive fullscreen)
- Русский язык в системе (Windows OCR для русского встроен в ОС)
